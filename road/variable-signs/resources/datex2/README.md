# Datex II XSD Schemas

These schemas are **reference material only** — they are not loaded, deployed, or used for validation anywhere in the application code.

Their purpose is to allow developers to:
- Inspect the Datex II data model and element structure
- Validate sample XML documents locally during development
- Look up type definitions when writing parsers

Each subdirectory contains the schema set for a specific Datex II version. See the `README.md` in each subdirectory for instructions on how to regenerate the schemas.
